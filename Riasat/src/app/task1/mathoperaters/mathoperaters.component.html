Lets get started with Basic Math Operaters

  • Binary


    • One operator between two operands. Just like a+b, where a and b are operands and '+' is operator.
    Other examples are:
    let a=10; let b=20 let sum = a+b; console.log(sum) //30;
    Example: string concatenation let a ="Hello" let b = "World" let c = a+ " "+b console.log(c) //"Hello World"
    Examples like a+b-c also comes under the binary.
    In these commands,
    We simply apply D-MAS to two or more operands to display result in other operand.

  • Unary Operators


    • These operators operate or work on single operators. Just like x--,
    Examples are: This is the increment operator that increases the value by 1 Example:
    let a = 10; a++; console.log(a); //11
    ++a; console.log(a); //12

  • Logical Operators


    • Logical operators include &&, ||
    These operaters are very useful when we have to validate more than one conditions.
    Example: let a=10 let b=15 console.log(a > 5 && b > 5) //true

  • Relational Operators


    • Check the values of the two operands and valiadates whether equal or not. And also It converts value to its conversable type. Basically it does not take type seriously. Operators are (==, !=,>,<,<=)
    Example:
    let a = 10; let b = 10; let c = "10"; console.log(a==b); //true console.log(a==c); //true
    Strict Equality Operator
    let a = 10; let b = 10; let c = "10"; console.log(a===b); //true console.log(a===c); //false It checks for the data types as well.
    More Examples are
    let a = 5; let b = 2; console.log(a >= b); //true console.log(a >= 4); //true console.log(a >= 5); //true console.log(a >= 6); //false
    let a = 5; let b = 2; console.log(a < b); //false console.log(a < 4); //false console.log(a < 5); //false console.log(a < 6); //true
