    • Enum Example: speed, slow, medium, fast;
    We mainly use enum where we have more than two options to select from. Also we have to assign values to enum attributes to access them easily. like slow: 1; medium:2; and so on.
    Other examples are: let go: direction; go= direction.up; go=direction.down;

    • When you don't have any idea of datatype or you are confused between two, so choose this.

    • Kind of function. Used to denote the no return value functions.

    • WE just validate that whether that object is empty or not. If not then add the actual value

  • Undefined

    • When any variable is not assigned to any value it will remian as undefined.

  • Objects

    • Object can not be of any data type. It is used to call classes. Each class has its unique identifier and class attributes can be called from that object only.
